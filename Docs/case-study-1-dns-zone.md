#Case Study 1:Azure Private DNS Zones

1. Global VNet Design
Case Study:  
Learnutec Global expands into East US and West Europe. Each region hosts Web, App, and Database tiers. The CTO requires secure communication, cross‑region failover, and isolation of the database tier.  
Tasks:  
- Create VNets in both regions with non‑overlapping address spaces.  
- Define subnets for Web, App, and DB tiers.  
- Configure VNet peering between regions.  
- Apply NSGs to enforce tier isolation.  
- Test connectivity with VMs.
